# Software Engineer Portfolio

This repository contains the source code for my personal portfolio website. It's a modern, single-page application built with React, TypeScript, and Tailwind CSS, designed to showcase my skills, professional experience, and technical projects to potential employers and collaborators.

## âœ¨ Features

- **Responsive Design:** Fully optimized for a seamless experience on desktop, tablet, and mobile devices.
- **Interactive UI:** Features smooth scrolling, subtle animations, and a clean, modern aesthetic.
- **Dynamic Content:** Data for experience, skills, and projects is centralized in `constants.tsx` for easy updates.
- **Comprehensive Sections:** Includes dedicated sections for:
  - **Hero:** A compelling introduction.
  - **About Me:** A brief professional bio.
  - **Experience & Development:** A timeline of my professional roles and self-directed learning.
  - **Technical Skills:** A categorized overview of my tech stack.
  - **My Work:** An interactive, multi-view carousel to showcase key projects.
  - **Contact:** A clear call-to-action to get in touch.
- **Performant Carousel:** The project showcase uses a performant, auto-scrolling carousel built with CSS Scroll Snap for a native-like feel, complete with navigation controls and dots.

## ğŸš€ Getting Started

To run this project locally, follow the steps below.

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or later recommended)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)

### Installation & Setup

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/himanshulade/your-portfolio-repo.git
    ```

2.  **Navigate to the project directory:**
    ```bash
    cd your-portfolio-repo
    ```

3.  **Install dependencies:**
    ```bash
    npm install
    # or
    yarn install
    ```

4.  **Start the development server:**
    ```bash
    npm run dev
    # or
    yarn dev
    ```

5.  Open your browser and navigate to `http://localhost:5173` (or the port specified in your terminal).
